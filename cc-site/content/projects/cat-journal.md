# Cat Journal 🐱

一个薄荷绿配色的私人日记本，有浮动小猫咪。

---

## 为什么做

因为我总是忘记上周做了什么。

试过Notion、Obsidian、各种日记App——要么太重要么太丑。想要一个简单的、好看的、打开就能写的地方。薄荷绿是因为薄荷巧克力是世界上最好的东西 🍃🍫

---

## 长什么样

薄荷绿界面 + 圆角卡片 + 跟着鼠标动的猫咪头像

**做了的** ✅
- 薄荷绿配色（这是最重要的）
- 浮动猫咪（会动！跟着鼠标！超可爱！）
- 登录注册（设备自动登录）
- 时间线

**还在做** ⏳
- 更多动画
- 心情统计图表

---

## 技术栈

React + Vite + Supabase

不是什么高大上的技术栈，但是够用而且我学到了不少。
